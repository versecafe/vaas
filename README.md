## Get back clean JSON Data

```json
[
  {
    "key": "2024-03-24",
    "views": 32,
    "visitors": 24
  },
  {
    "key": "2024-03-25",
    "views": 183,
    "visitors": 72
  },
  {
    "key": "2024-03-26",
    "views": 569,
    "visitors": 216
  },
]
```

### Run it yourself

Vaas has no env vars, so all you need to do is clone the repo and run the following commands:

```bash
bun i && bun dev
```
